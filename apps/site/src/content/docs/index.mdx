RunMyCrew turns plain-English intent into running automation. Describe what you
want, connect the apps you already use, and every execution is logged
end-to-end — no glue code, no cron server to babysit.

## Overview

Workflows in RunMyCrew are graphs of **triggers**, **conditions** and
**actions**. They run on schedules, webhooks or app events, and every run is
captured with its inputs, outputs, and timing so you can see exactly what
happened.

## Core concepts

- **Triggers** — start a workflow from an app event, a webhook, or a schedule.
- **Conditions** — branch and filter without writing glue code.
- **Actions** — fan out to every connected tool in a single run.
- **Connections** — OAuth-managed credentials for each integration.

## Install

Self-host with Docker:

```bash
docker compose -f deploy/docker-compose.production.yml up -d
```

Or sign up for the hosted app and skip the infrastructure entirely — it's free
while we're in [early access](/pricing).
